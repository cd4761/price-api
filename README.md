# RESTful API for TON price

```
$ npm install
$ node app.js
```

### API List
| ROUTE                     | METHOD | DESCRIPTION               |
|---------------------------|--------|---------------------------|
| /api/price                | GET    | 모든 price 데이터 조회        |
| /api/latest               | GET    | 최신 price 데이터 조회        |
